# snake-game
**Historic Snake game made using Pygame**


Pygame must be installed before running the code. It can be installed by running:

`pip install pygame`

**To run this code locally:**

`git clone https://github.com/debjit-mandal/snake-game`

`cd snake-game`

`python main.py`
